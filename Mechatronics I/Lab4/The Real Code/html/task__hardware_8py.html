Detailed Description

Task to assign motors and encoders to hardware pins, checking for user to disable faults, and setting duty cycle to shared value.

Sets motors/encoders to their repective pins and channel numbers then enables the driver. A controller is instantiated to be used to control duty based on a current and reference velocity. Tasks constantly updates shared values and checks for flags executed in task user.
