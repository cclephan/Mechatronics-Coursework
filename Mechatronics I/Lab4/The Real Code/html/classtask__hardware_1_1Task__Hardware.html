◆ nextUpdate()

def task_hardware.Task_Hardware.nextUpdate (   self)

Updates encoder position, velocity, and time then sets duty to what is calculated in controller update method.

Uses encoder driver to update position/velocity from which the values are written into the MotorShare list and the period is updated. Duty is determined by controller update method, which takes in reference/current velocity and a time difference.

def task_hardware.Task_Hardware.run (   self)

Updates shared values of current/reference position and velocity.

Detects for UI flags and if PID gains change.

Constantly updates motor position and velocity and executes logic for flags sent from task_user such as zeroing position or reseting velocity and PID gains when a fault is disabled.
