Detailed Description

This file contains functions and a test scrip to fluctuate LED brightness.

Functions within this file set up an LED pin on the Nucleo and fluctuates the PWM. The test script uses these two functions to create a sawtooth pattern for LED brightness.The user can end the program by pressing control+c.
