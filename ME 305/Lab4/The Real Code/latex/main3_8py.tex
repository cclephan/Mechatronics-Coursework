\doxysubsection{Detailed Description}
Main file, which hardeware and user task are being run. 

File imports the classes for each task and calls out the period at which they will be run. The tasks are created as objects and share objects containing information shared to operate motor. Additionally a motor driver is created to be called as a parameter in motor task. A while loop is created to constantly run through tasks at their frequency specified by period variables. The user can press cntrl+c as a keyboard interupt to stop the program. See Results/\+Discussion page for data collection. ~\newline
