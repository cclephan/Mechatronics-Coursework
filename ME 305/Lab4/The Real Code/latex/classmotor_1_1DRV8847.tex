\doxysubsection{Detailed Description}
A motor driver class for the \mbox{\hyperlink{classmotor_1_1DRV8847}{DRV8847}} from TI. 

Objects of this class can be used to configure the \mbox{\hyperlink{classmotor_1_1DRV8847}{DRV8847}} motor driver and to create one or more objects of the \mbox{\hyperlink{classmotor_1_1Motor}{Motor}} class which can be used to perform motor control. Refer to the \mbox{\hyperlink{classmotor_1_1DRV8847}{DRV8847}} datasheet here\+: \href{https://www.ti.com/lit/ds/symlink/drv8847.pdf}{\texttt{ https\+://www.\+ti.\+com/lit/ds/symlink/drv8847.\+pdf}}
